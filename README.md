# PRODIGY_DS_03

## Decision Tree Classifier using Bank Marketing Dataset

This project builds a machine learning model using a Decision Tree Classifier to predict whether a customer will subscribe to a term deposit.

## Dataset
Bank Marketing Dataset from UCI Repository

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Steps Performed
1. Imported dataset
2. Preprocessed data
3. Encoded categorical variables
4. Split data into training and testing sets
5. Trained Decision Tree Classifier
6. Evaluated model performance
7. Visualized decision tree

## Output
- Accuracy Score
- Classification Report
- Decision Tree Visualization

## Learning Outcome
Learned how to build and evaluate a supervised machine learning classification model using Decision Trees.
